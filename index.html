Following are the commands which I learned to create this Webpage:

GIT

Git init = To create the repository
Git Add filename = To add the file(s) to git in preparation for a commit
Git commit -m “message”= To do the commit to save changes
Git checkout -b branch name = To make a new branch and checkout
Git checkout branch name = To checkout branch
Git merge BRANCHNAME = To merge the branch Note:You have to be on the branch that you want to merge to

LINUX

Cd - To change directory
Mkdir Name - To make directory
ls - To list file
vi filename - To entering into vi for the specific file

VI

I = To enter into insert mode
Escape = To get out of insert mode
Shift key + colon(:) = To exit for vi
W = Command to write
Q = Command to quit
WQ = Write and Quit
